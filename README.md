# Tetris HTML Game

## 概要

`テトリス.html` は、HTML、CSS、JavaScript だけで実装されたシンプルなテトリスゲームです。

- 10 列 × 20 行の標準的なテトリスボード
- 7 つのテトロミノ形状をランダムに生成
- ブロック移動、回転、下方向移動の操作をサポート
- スコア表示と次のブロック表示を搭載
- モバイル向けタッチボタンも実装

## ディレクトリ構成

- `テトリス.html` — ゲーム本体
- `README.md` — 設計書および操作マニュアル

## 設計書

### 1. ゲーム構造

- HTML
  - `canvas#gameCanvas` — メインゲームフィールド表示用
  - `canvas#nextCanvas` — 次のブロック表示用
  - `button#startButton` — ゲームスタートボタン
  - タッチ操作ボタン（左右・回転・下移動）

- CSS
  - ゲーム画面のレイアウト
  - ダークテーマのビジュアルスタイル
  - レスポンシブ対応（画面幅 600px 以下で縦並び表示）

- JavaScript
  - ゲーム状態管理: `initial`, `playing`, `paused`, `gameover`
  - ボード生成: `createBoard()`
  - テトロミノ生成: `getRandomPiece()`
  - 描画処理: `drawBoard()`, `drawPiece()`, `drawNextPiece()`
  - 衝突判定: `checkCollision()`
  - 移動・回転: `movePiece()`, `rotatePiece()`
  - ロック処理: `lockPiece()`
  - ライン消去: `clearLines()`
  - スコア計算: `addScore()`
  - メインループ: `update()`

### 2. 主要データ構造

- `board`
  - 2 次元配列 `ROWS x COLS`
  - 0 は空セル、色文字列はブロックの存在を表す

- `SHAPES`
  - 各テトロミノの形状と色を定義
  - 7 つのテトロミノ: I, O, T, L, J, S, Z

- `currentPiece`, `nextPiece`
  - 現在操作中のブロックと次のブロックを管理
  - 各ピースは `shape`, `color`, `x`, `y` を持つ

### 3. ゲームフロー

1. `startGame()` によりゲーム開始
2. `board` を初期化し、スコアをリセット
3. `newPiece()` で最初のブロックを生成
4. `requestAnimationFrame(update)` でループ開始
5. 一定時間ごとに `dropPiece()` を呼び出しブロックを下降
6. ブロックが衝突したら `lockPiece()` で固定
7. `clearLines()` で消去とスコア更新
8. 新しいピースを生成し、ゲームオーバー判定

### 4. 衝突判定の仕組み

`checkCollision()` は現在のピース形状をボード上の位置に重ねて判定する。

- 横方向がボード外の場合: 失敗
- 下方向がボード外の場合: 失敗
- ブロックが既存のセルと重なる場合: 失敗

### 5. 回転処理

- 90 度回転は行列の転置と反転で実装
- 衝突時は元の形状に戻す

### 6. スコア

- 1 行消去: 100 ポイント
- 2 行消去: 300 ポイント
- 3 行消去: 500 ポイント
- 4 行消去: 800 ポイント

### 7. UI と操作

- キーボード操作
  - `ArrowLeft`: 左移動
  - `ArrowRight`: 右移動
  - `ArrowDown`: 1 マス落下
  - `ArrowUp`: 回転

- タッチ操作
  - `◀` 左移動
  - `↻` 回転
  - `▶` 右移動
  - `▼` 下移動

- 画面下部のタッチボタンはモバイル表示に対応
- `ゲームスタート` ボタンでプレイ開始
- ゲームオーバー後は「もう一度プレイ」ボタンで再スタート

## 操作マニュアル

### ゲームの開始

1. `テトリス.html` をブラウザで開きます。
2. 画面中央の `ゲームスタート` ボタンをクリックします。
3. ゲームが開始し、スコアが `0` にリセットされます。

### ゲーム操作

- キーボードで操作する場合:
  - `←` でブロックを左へ移動
  - `→` でブロックを右へ移動
  - `↓` でブロックを 1 行落とす
  - `↑` でブロックを回転

- タッチ操作の場合:
  - `◀` ボタンで左移動
  - `↻` ボタンで回転
  - `▶` ボタンで右移動
  - `▼` ボタンで下移動

### 目的

- ブロックを横一列に並べてラインを消去します。
- 消去したライン数に応じてスコアが加算されます。
- ブロックが画面上部に積み上がって新しいピースを置けなくなるとゲームオーバーです。

### ゲームオーバー

- ブロックを固定した際に、次のピースが生成される位置で衝突するとゲーム終了。
- ゲームオーバー画面が表示され、スコアが確認できます。
- `もう一度プレイ` ボタンを押すと再スタートします。

## 実行方法

1. `TetrisRepo` フォルダに移動
2. `テトリス.html` をブラウザで開く

例:

```powershell
Set-Location -Path "d:\ChatGPT\TetrisRepo"
Start-Process .\テトリス.html
```
